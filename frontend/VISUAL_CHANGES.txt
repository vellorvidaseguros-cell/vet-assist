╔══════════════════════════════════════════════════════════════════════════════════════════════════════════════════╗
║                                      FLUXO DE USO                                                               ║
╚══════════════════════════════════════════════════════════════════════════════════════════════════════════════════╝

1. Usuário abre aba "Perfil" → Vê card "Tabela de Preços" com botão limpo

2. Clica em [⚙️ Gerenciar Tabela de Preços] → Modal abre
   - Desktop: Modal centralizado (max 600px)
   - Mobile: Bottom-sheet desliza de baixo (100% width)

3. Modal mostra lista de serviços com:
   - Nome do serviço
   - Valor padrão
   - Input para valor customizado
   - Botão 🗑️ para remover

4. Usuário pode:
   A) Editar valor customizado de serviço existente
      → Digita novo valor no input

   B) Remover serviço
      → Clica 🗑️ → Confirmação

   C) Adicionar novo serviço
      → Clica "+ Adicionar Serviço"
      → Form abre inline
      → Preenche nome e valor
      → Clica "✓ Adicionar"

5. Clica [💾 Salvar Tabela] → Backend recebe todos os dados

6. Modal fecha e retorna à Perfil com sucesso!
